# Vinecraft – Block Reference

All block types currently in the game, their properties, and where to find them.

## Block Properties Key

| Property | Meaning |
|---|---|
| Hardness | Seconds to break with bare hand |
| Tool | Best tool type for fast breaking |
| Drops | What item drops when broken |
| Luminance | Light level emitted (0 = none, 15 = max) |
| Opaque | Whether light passes through |

## Block Breaking Times (Bare Hand Reference)

Breaking time varies significantly by material and tool. Always use the recommended tool for faster breaking:

- **Shovel** fastest on: Dirt, Grass, Sand, Gravel, Snow, Clay, Soul Sand
- **Pickaxe** fastest on: Stone, Ore, Brick, Cobblestone, Marble, Crystal, Ice
- **Axe** fastest on: Wood, Log, Planks, Bookshelf, Pumpkin, Melon

Using the wrong tool adds a **5× time penalty**.
