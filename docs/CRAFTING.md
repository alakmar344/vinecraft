# Vinecraft – Crafting Guide

## How Crafting Works

Open the inventory with **E**. The bottom of the inventory screen shows a **2×2 crafting grid**. Place items in specific patterns to create new blocks and tools.

> **Tip:** Crafting tables (placed in the world) unlock **3×3 recipes** for more complex items.

---

## 2×2 Recipes (Inventory Crafting)

These can be crafted anywhere using the inventory crafting grid.

### Wood Planks
```
[Log] [Log]       →  4× Wood Planks
[Log] [Log]
```

### Sandstone
```
[Sand] [    ]     →  1× Sandstone
[Sand] [    ]
```

### Stone (Smelting shortcut)
```
[Cobble] [Cobble]  →  4× Stone
[Cobble] [Cobble]
```

## Resource Progression

```
Trees → Logs → Planks → (crafting table) → Tools, Chests
Stone + Ore → Pickaxe → Deeper Mining
Clay → Bricks / Terracotta (via crafting)
Crystal → Prismarine (unique material)
Sand + Gravel → TNT
```

## Tips

1. Always craft **planks** first — they unlock everything else.
2. **Glowstone** is the best light source; find it or craft from Crystal.
3. **Terracotta** and **Marble** make beautiful builds.
4. **TNT** can be used for mining tunnels quickly.
5. Stack up on **Cobblestone** early — it's the most versatile building block.
